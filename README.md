# Golang 土拨鼠项目爬虫示例

## 使用
1. 克隆`https://github.com/hunterhug/GoSpider`仓库到GOPATH路径下，地址：[https://github.com/hunterhug/GoSpider](https://github.com/hunterhug/GoSpider)
2. 示例仅供学习，爬虫有风险，如果太暴力，会给别人带来损失，在此申明不承担相应责任，查看Example
